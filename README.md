# Oneline_Retail_-Nettoyage_Python_Dashboard_PowerBY
Ce tableau de bord interactif permet d’analyser les performances commerciales d’un site e-commerce à partir de données transactionnelles. Il met en évidence les indicateurs clés tels que le chiffre d’affaires, le volume de commandes, la fidélité client et la répartition géographique.
---
## Données utilisées
Source : Online_Retail_clean.csv
Période : 2010–2011
Colonnes principales : InvoiceDate, CustomerID, Country, Description, Quantity, UnitPrice
Champs dérivés : CA, NbCommandes, NbClients, RatioFidelite, Month, year
---
## Résultats clés
Forte concentration des ventes sur quelques produits
Croissance continue du CA entre Q1 et Q4 2011
Répartition internationale des clients
